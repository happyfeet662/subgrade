書き出し初心者向け

書き出しの超入門
迷わずYouTube用のMP4にする

編集した内容は、そのままではただの作業データ。1本の動画ファイルに変換して、はじめてYouTubeやSNSに上げられます。これが書き出し（エクスポート）。 設定項目は多くて不安になりますが、初心者は「H.264 ＋ YouTubeプリセット」を選ぶだけでOK。最短ルートだけ覚えましょう。

この記事のゴール ①ファイル > 書き出し > メディア → ②形式「H.264」→ ③プリセット「YouTube 1080p」系 → ④保存先を決めて書き出し。 たったこれだけで、どこにでもアップできるMP4ができます。

まず用語：プロジェクト ≠ 動画ファイル

Premiere で保存されるプロジェクト（.prproj）は「編集の設計図」で、これ自体は動画ではありません。 YouTubeに上げたり友達に送ったりできるMP4などの動画ファイルを作る作業が「書き出し」です。ここを混同しないようにしましょう。

最短の手順

  1. タイムラインを選んだ状態で ファイル > 書き出し > メディア（ショートカット Ctrl/Cmd + M）。
  2. 形式で H.264 を選ぶ（＝MP4になる、最も汎用的）。
  3. プリセットで 「Match Source – 高速ビットレート」 か 「YouTube 1080p Full HD」 を選ぶ。
  4. 出力ファイル名と保存先を決める。
  5. 書き出しボタンを押す。完了するとMP4ができ上がり。

「Match Source（ソースに合わせる）」を選ぶと、編集した解像度・フレームレートに自動で合わせてくれるので、初心者はまずこれが安全です。

迷ったらこの設定（早見）

項目初心者の選択
形式H.264（＝MP4）

解像度編集どおり（1080p など）。基本いじらない
音声プリセット任せでOK（AACになる）

ビットレートやVBRなど細かい数値は当面いじらなくて大丈夫。画質を追い込みたくなったら、書き出し設定の正解（中級）に早見表があります。

「書き出し範囲」に注意

初心者のいちばんの事故が「一部しか書き出されていない」。タイムラインにイン点・アウト点（範囲指定）が残っていると、その区間だけが書き出されます。 書き出し画面の「範囲」が「シーケンス全体」になっているか、必ず確認しましょう。

つまずきポイント ・一部しか書き出されない/短い → 範囲が「イン/アウト」になっている。「シーケンス全体」に。
・映像が真っ黒 → 調整レイヤーやフェードの黒が残っていないか、範囲がズレていないか確認。
・音が出ない → 音声トラックがミュート(M)になっていないか、書き出し設定で音声がオンか。
・ファイルが重すぎ/時間がかかる → プリセットが4Kや高ビットレートかも。1080pのYouTubeプリセットに。

よくある質問

結局いちばん無難な設定は？

形式「H.264」＋プリセット「Match Source」または「YouTube 1080p」。これでYouTube・Instagram・TikTokどこにでも上げられるMP4になります。

「メディアエンコーダーで書き出し」って何？

書き出しを別アプリ(Adobe Media Encoder)に渡して、その間もPremiereで作業を続けられる機能です。急がないなら普通の「書き出し」でOK。複数本まとめて出すときに便利です。

H.265の方が軽いと聞いたけど？

H.265は容量が小さい反面、古い環境で再生できないことがあります。不特定多数に配るならH.264が安全。使い分けは書き出し設定の正解（中級）で。

次に読むべき記事は？

画質を詰めたいなら書き出し設定の正解（中級）、色を整えるなら色調整の入門、音は音の調整の基本へ。

まとめ

書き出しは「Cmd+M → H.264 → Match Source / YouTube 1080p → 範囲がシーケンス全体か確認 → 書き出し」。 細かい数値は後回しでかまいません。まずは「最後まで正しく1本のMP4にする」を確実に。これで、作った動画をやっと世に出せます。
